## Installation
1. Install ocaml raylib bindings using 

    ```sh 
    opam install raylib 
    ```
2. Install yojson using

    ```sh 
    opam install raylib 
    ```

## Usage
1. Initiate build process.

    ```sh 
    make build
    ```
2. Run the game.

    ```sh 
    make game
    ```
3. Use A,D for movement, space for jumping, and j,k,l,u for attacks
